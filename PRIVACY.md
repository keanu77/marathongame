# 隱私說明

最後更新：2026 年 7 月 12 日

本文件描述此儲存庫目前實作與官方示範部署的資料行為。自行 Fork、修改或部署的人可能採用不同設定、服務或保存政策，應自行提供符合其實際情況的隱私說明。

## 摘要

- 不需要建立帳號，也沒有廣告或應用程式內建的第三方分析追蹤碼。
- 瀏覽器會在 `localStorage` 保存本機最高分；清除該網站的瀏覽資料即可移除。
- 開始遊戲後會自動建立網路驗證跑局，並約每 10 秒傳送遊戲進度、收集道具數與遊戲內體力／受傷風險；即使最後不加入排行榜，這些驗證資料仍會依下文方式處理。
- 玩家只有在結算後主動輸入暱稱並送出時，成績才會寫入 Cloudflare D1 的跨裝置排行榜。
- 排行榜暱稱與部分遊戲成績會公開顯示。請勿使用真實姓名、聯絡方式、病歷或其他敏感資訊作為暱稱。
- 目前沒有帳號綁定、自助修改或自助刪除網路成績的功能，網路紀錄也沒有自動到期政策。

## 瀏覽器內資料

目前遊戲使用 `localStorage` 鍵 `marathon-prep-runner.high-score.v2` 保存這個瀏覽器中的最高分。資料留在該瀏覽器與該網站來源下，不會因最高分功能本身上傳到伺服器；無痕模式、瀏覽器權限或儲存空間限制可能使它無法保存。

程式碼亦保留 `marathon-prep-runner.local-leaderboard` 的本機排行榜儲存 helper，供測試或未來／衍生版本使用；目前正式跨裝置排行榜流程以 D1 為準，並未把本機排行榜作為網路送分資料來源。

你可以用瀏覽器的「清除網站資料」功能刪除這些本機資料。遊戲目前未提供只刪除單一 localStorage 鍵的介面。

## 網路排行榜資料

每局開始時，伺服器建立一次性跑局 ID 與 token。token 會交給瀏覽器供該局驗證；D1 只保存 token 的 SHA-256 雜湊，不保存可直接重用的原始 token。

### 跑局與檢查點

D1 會處理或保存：

- 不透明的跑局 ID、規則版本、發行／到期時間與跑局狀態；
- 約每 10 秒提交的遊戲有效時間、已收集道具數、遊戲內體力、遊戲內受傷風險與檢查點時間；
- 結算時間及用於辨識重複提交的結算指紋；
- token 雜湊，而非原始 token。

上述體力與受傷風險是遊戲內虛構狀態，不是穿戴裝置、生理量測、診斷或真實醫療紀錄。本遊戲不要求玩家輸入病史、症狀、年齡或真實訓練資料。

### 成績資料

玩家主動送分時，D1 另保存：

- 玩家輸入的暱稱；
- 分數、距離、有效遊戲時間、收集道具數；
- 完賽或中途停止、抵達階段；
- 終點遊戲內體力、終點遊戲內受傷風險、健康完賽加分；
- 不透明的跑局／成績 ID 與建立時間。

公開排行榜 API 目前回傳前 10 名的成績 ID、暱稱、分數、距離、結果、階段與健康完賽加分；終點體力與終點受傷風險不直接列在公開榜單中。送分完成時，該玩家的瀏覽器也會收到自己的成績與是否進入前 10 名。榜單會隨其他玩家成績而變動。

## 網路與濫用防護資料

為限制大量建立跑局，Cloudflare 會將連線 IP 提供給 Pages Functions。應用程式以伺服器 secret、UTC 日期與 IP 產生每日 HMAC 雜湊鍵，D1 的速率限制表只保存該雜湊鍵、路由、時間窗與計數，不保存原始 IP。

Cloudflare 作為代管與資料庫服務提供者，仍會處理網路請求所需的 IP、HTTP 標頭及營運／安全資訊；其處理與保存也受 Cloudflare 帳戶設定與 Cloudflare 自身政策影響，不由此儲存庫的程式碼完整控制。

## 使用目的

上述資料只用於：

- 保存與顯示跨裝置排行榜；
- 由伺服器重算分數、驗證合理進度與處理重試；
- 限制自動化濫用並排查服務錯誤；
- 在玩家主動操作時產生成績分享文字或圖片。

本實作沒有以玩家資料投放廣告，也沒有建立真實世界健康風險或個人醫療檔案。

## 公開性與第三方

- 進入公開榜單的暱稱與成績資訊，任何能存取排行榜 API 的人都可能看見、截圖或另行保存。
- 玩家按下分享後，遊戲會優先呼叫瀏覽器的 Web Share API；若使用者選擇社群、通訊或其他目標服務，資料會依該服務的隱私政策處理。無法分享時可能複製文字到剪貼簿，或由玩家下載成績圖片。
- 衛教資料來源連結會前往外部網站；只有在玩家點擊後才會離開遊戲，後續資料處理由該網站決定。
- 正式網站與 D1 目前由 Cloudflare 提供服務。

## 保存期間與刪除限制

目前程式的實際行為如下：

- 本機最高分會保留到玩家清除網站資料、瀏覽器自行移除資料，或程式日後更換儲存鍵為止。
- 尚未送出的過期跑局會在後續有人建立新跑局時，機會性清理「已過期超過約 24 小時」的紀錄；這不是精準排程，因此可能保留更久。
- 速率限制計數會在後續建立跑局時，機會性清理約兩天以前的時間窗；同樣可能因沒有新請求而保留更久。
- 已送出的跑局、檢查點稽核資料與排行榜成績目前沒有自動到期或定期刪除機制，會保存到維護者手動清理、重設賽季或變更政策為止。

因為沒有帳號，本遊戲無法可靠證明某個暱稱或匿名成績屬於哪一位自然人，也沒有自助刪除／改名 API。若要詢問移除網路成績，請透過專案擁有者在 GitHub 公開的聯絡方式或[製作者網站](https://sportsmedicine.tw/)先取得私人聯絡管道，並提供足以定位紀錄的暱稱、分數與大約送出時間；不要在公開 Issue 張貼敏感資訊。是否能精準定位及驗證請求者身分有實作上的限制，目前只能由資料庫管理者手動處理。

## 你的選擇

- 不送出暱稱與成績，仍可完成遊戲；只是無法加入跨裝置排行榜。
- 使用不含真實身分或健康資訊的暱稱。
- 透過瀏覽器設定清除本機最高分。
- 不使用分享按鈕，或在作業系統分享面板出現後取消。
- 若不同意 Cloudflare 處理排行榜請求，可只使用純前端本機遊戲，不送出網路成績；自行部署者也可移除 Functions／D1 功能。
